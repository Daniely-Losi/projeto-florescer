# Florecer - Confirmação de Presença

Este projeto é um site para confirmar a presença em um encontro de mulheres organizado pelo Projeto Florecer. O site fornece informações sobre o evento e permite que as participantes confirmem sua presença de forma rápida e fácil.

## Funcionalidades

- **Confirmação:** Permite que as participantes confirmem sua presença.
- **Detalhes do Evento:** Inclui data, hora e localização do encontro.
- **Design Responsivo:** Compatível com dispositivos móveis e desktops.

## Tecnologias Utilizadas

- **HTML**
- **CSS**
